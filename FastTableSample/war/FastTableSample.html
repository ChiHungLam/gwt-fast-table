[Beta] Experimental API to build full-featured, fast-rendering large tables, while providing full access to the underlying domain objects and DOM elements
When the page is loaded, 100 simple domain objects are generated and displayed.

To evaluate performance, sort by clicking a column header, or experiment with the row-count and build rows incrementally options. Performance varies by browser: Chrome is excellent, IE7/8 are poor.

Sorting demonstrates cell-level event handling. Row-level event handling is demonstrated by highlighting on mouse over, or by selecting a row.

Selecting a row opens a drop panel, which demonstrates that we have access to the underlying domain object and DOM structure. Clicking the drop panel "OK" button exhibits the smooth integration of "setInnerHtml()" and standard GWT widget technologies.
