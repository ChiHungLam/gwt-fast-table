[Beta] Experimental API to build full-featured, fast-rendering large tables, while providing full access to the underlying domain objects and DOM elements
When the page is loaded, 1000 simple domain objects are generated and displayed. To evaluate performance, sort by clicking a column header. To demonstrate cell-level event handling, mouse over "Name" or "Zip" cells. Select a row by clicking "Name" or "Zip" -- the resulting drop panel demonstrates that we have access to the underlying domain object and DOM structure. Clicking the drop panel "OK" button exhibits the smooth integration of "setInnerHtml()" and standard GWT widget technologies.
